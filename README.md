# 📰 A Hora é Agora — Portal de Notícias Financeiras

Projeto de um **site de notícias sobre finanças, economia e investimentos**, desenvolvido como parte do portfólio de estudos em desenvolvimento web.

O objetivo do projeto é demonstrar conhecimentos em **HTML, CSS e JavaScript**, criando um portal informativo com layout moderno e navegação entre páginas.

---

# 🌐 Sobre o Projeto

O site **A Hora é Agora, comigo Cauê Serra** foi criado para simular um **portal de conteúdo financeiro**, com notícias e informações sobre:

* Economia
* Investimentos
* Mercado financeiro
* Criptomoedas
* Educação financeira

O projeto também demonstra a estrutura básica de um **site institucional com múltiplas páginas**.

---

# 🚀 Tecnologias Utilizadas

* **HTML5** → Estrutura das páginas
* **CSS3** → Estilização e layout responsivo
* **JavaScript** → Interatividade e manipulação do DOM

# 🎨 Interface e Design

O site possui:

* Layout limpo e moderno
* Cards de notícias com **efeito hover**
* Menu de navegação no topo
* Estrutura semelhante a portais de notícias

O CSS utiliza:

* **Flexbox**
* **CSS Grid**
* **Media Queries** para responsividade

---

# ⚙️ Funcionalidades JavaScript

O arquivo **script.js** adiciona interatividade ao site.

Funcionalidades implementadas:

✔ Detecção do carregamento da página
✔ Registro de navegação nos links do menu
✔ Base preparada para futuras integrações com APIs

Exemplo de funcionalidade:

```javascript id="lj29ti"
document.addEventListener('DOMContentLoaded', () => {
    console.log('Site carregado com sucesso!');
});
```

O script também permite expandir o projeto futuramente para:

* buscar notícias automaticamente
* integrar APIs financeiras
* adicionar novas funcionalidades dinâmicas
